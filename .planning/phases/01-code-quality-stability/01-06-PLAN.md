Note on clearing `focusOnMount`: The simplest approach is to fire `data.onEdit?.(data.id, {})` on mount when `focusOnMount` is true — this triggers `styledNodes` recomputation, which will recalculate `focusOnMount` as undefined on the next render (since it comes from `addTaskNode` and is not stored in nodes state permanently). Actually the cleanest approach: do NOT call onEdit to clear it — just focus once using `useEffect([], [])` with an empty dep array so it only fires on initial mount. The `focusOnMount` in node data won't cause re-focus because the useEffect has empty deps.
